# turtlecoin.lol

To submit or propose changes to the main marketing page, submit a pull request to this repository


## Getting Started

### Prerequisites

[Node.js](https://nodejs.org/en/)


Gulp
```
npm install -g gulp
```
### Build Instructions

Clone the repository.
```
git clone https://github.com/turtlecoin/turtlecoin.lol
cd turtlecoin.lol
```
Install packages
```
npm install
```
Run tests
```
gulp test
```
Build
```
gulp build
```
