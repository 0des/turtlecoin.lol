烏龜幣有趣的特性

烏龜幣每30秒產生一個區塊，這代表轉帳交易更加快速。

烏龜幣的區塊鍊保障您的隱私，除了使用者之外，沒有人能檢視您的資金。

您可以利用 view key 及 spend key 復原您的烏龜幣錢包。

為何選擇烏龜幣?

1. 快速的轉帳交易

烏龜幣每30秒產生一個區塊，相較於比特幣或比特金每10分鐘產生一個區塊，烏龜幣的轉帳快了20倍。

2. 保障隱私

烏龜幣與門羅幣以及永世幣同樣具有保障隱私的特性，除非您決定公開交易，不然每一個轉帳交易都是隱蔽的。

3. 容易使用

我們支持幾乎每一個作業系統，即便在行動裝置上，您也可以免費 產生 一個安全的紙錢包，從建立錢包到開始使用只需不到5分鐘。

4. 容易開採

烏龜幣本身自帶 CPU 挖礦程式，但您也可以使用任何您熟悉的門羅幣挖礦程式或使用 GPU 在礦池挖礦。我們推薦使用 XMR-Stak Unified Miner.

5. 龐大的社群

烏龜幣的社群歡迎所有的使用者或開發人員。 你不會因為犯錯而被社群責備，而且我們歡迎任何對烏龜幣的批評與建議。請加入我們的 Discord Chat。

6. 支持

我們正在 GitHub meta-forum 上塑造一個開發人員及測試人員的社群。您可以藉由測試軟體、提供除錯報告以及鼓勵來協助我們。

故事

我們的背景

大多數的烏龜幣開發人員都有厚重且賞心悅目的鬍子。 這是真的。

。

這個特徵讓我們 在撫摸下巴這個技能上多了10個點的經驗值 ，並且強化我們的耐心及判斷力。


事實上，大多數的烏龜幣開發人員都有其他的全職工作，而烏龜幣是提供我們樂趣並回饋給社群的一個方式。

烏龜幣是如何產生的?

在12月、一個寒冷的夜晚，駭客 RockSteady 及 Bebop 兩人喝了太多啤酒，並且討論到世界上有太多造假、而實際上沒有產品開發實力的區塊鍊項目ICO。Bebop 認為在這個時代，若沒有認真的團隊、認真的行銷、以及正式的經費來源，不可能建立一個有用的區塊鍊項目。

RockSteady 熱切地摸著鬍子，仔細地聆聽 Bebop。他只有一個反應..

"拿好我的啤酒，我做給你看"

這個夜晚，2017年12月9日，烏龜幣誕生了。

為何創造烏龜幣?

許多的區塊鍊項目獲得了 太多的經費支持 而大多數 根本沒有實際產出。我們理解到一點 "為何不讓整個事情倒著來?" 發起一個好玩的區塊鍊項目，讓它儘可能的容易使用且容易獲得、讓它自己成長。

我們強烈地相信我們的無經費策略；如果我們做出好的產品， 產品會自己創造價值。 如果你認為我們是錯的，我們歡迎您來 談談 你的想法。

產品路徑圖

  • Q4 2017

    嗨! 烏龜

    烏龜幣誕生於 2017年12月9日.

    剛開始，烏龜幣只有命令列錢包，只能從 Linux 編譯。當時由於贈送烏龜幣，常常被網路論壇取締。

  • Q1 2018

    圖形介面的烏龜幣錢包誕生

    烏龜幣社群積極給使用者帶來圖形化的體驗，同時更新了命令列錢包。打從第一天開始，我們便承諾要讓烏龜幣變得更容易使用，而這得從圖形化開始。從這些錢包中，挑一個您喜歡的:

    • Golang GUI [Win/Mac/Lin]
    • Winforms GUI [Win/Mac/Lin]
    • Python GUI [Win/Mac/Lin]
    • Electron GUI [Win/Mac/Lin]
    • C# GUI [Windows]
  • Q2 2018

    行銷夥伴

    我們承諾讓烏龜幣在2018年的第二季能公開交易，而我們已經達成了，現在烏龜幣能在數個交易所進行交易。

  • 2018年的第二季，產品開發開始

    Karai，簡單的智能合約

    散布式且不可靠的網絡使用 beyond currency。開發 Karai 是我們的目標，它能在主要的區塊鍊上發布程式及網絡。

  • TBA

    Mobius，快速的區塊且無須同步

    同步巨大的區塊鍊是工作量證明機制一個令人頭痛的問題。快速的產生區塊並不會解決這個問題。我們的目標是在較短的鍊上更快地產生區塊。這是最有挑戰性的目標，因此我們需要更多的幫助。
